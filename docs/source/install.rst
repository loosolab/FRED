How to install
===============


Via Python package
--------------------

FRED is published in the Python Package Index (PyPI) under the name **`fred-metadata <https://pypi.org/project/fred-metadata/>`_**\ . It can be installed with the following command:

.. code-block::

    pip install fred-metadata


Via Repository
---------------------

You can install FRED directly from the repository with the following commands:

1. Clone the repository 

.. code-block::
    
    git clone https://github.com/loosolab/FRED.git


2. Move to the new repository directory

.. code-block::

    cd FRED


3. (Optional) Create and activate a pip environment

.. code-block::

    python3 -m venv .venv
    source .venv/bin/activate

4. Install the package

.. code-block::

    pip install .
